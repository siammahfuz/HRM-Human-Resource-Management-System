# HRM — Human Resource Management System

A modern and scalable HRM (Human Resource Management) System built to streamline employee management, attendance tracking, payroll processing, recruitment, performance evaluation, and organizational workflows. This system is designed with clean architecture, reusable modules, and database-driven operations using PHP and MySQL.

🚀 Features

📌 Core Features

* Employee & Department Management
* Attendance & Shift Management
* Payroll & Salary Processing
* Leave & Holiday Management
* Recruitment & Job Posting System
* Employee Performance Evaluation
* Task & Workflow Management
* User Role & Permission Management
* Employee Profile & Document Storage
* HR Dashboard & Reports
* Notice Board & Internal Communication
* Single Company Workspace
* Basic Reporting & Analytics

⚡ Advanced Features

* Automated Payroll Generation
* Biometric Attendance Integration
* Employee Self-Service Portal
* Recruitment Workflow Automation
* Performance KPI Tracking
* Training & Development Management
* Multi-user Collaboration System
* Custom HR Modules & Fields
* Live Notification & Alert System
* API Integration Support
* Employee Contract Management
* Advanced HR Analytics & Reports
* Activity Audit Trail
* Third-party Integration Ready

🛠️ Technologies Used

* PHP (Core / Laravel optional)
* MySQL Database
* HTML, CSS, JavaScript
* Bootstrap / Tailwind CSS (optional)
* AJAX / jQuery

* Import hrm_system.sql

2️⃣ Configure Database
/config/database.php

3️⃣ Run Project

* Start local server (XAMPP / Laragon)
* Open browser:
  http://localhost/hrm-system
